Europeana OAI-PMH Server

Developed by Ontotext Corp as part of the Europeana Creative project. Uses the OAICat Open Source implementation of OAI-PMH v2.0 and the Europeana MongoDB production databases. This demonstration of the interfaces is based on the webapp distribution and can be tested through the protocol request forms listed below.

Protocol Request Forms

  1. Identify
  2. GetRecord
  3. ListIdentifiers ( Resumption)
  4. ListMetadataFormats
  5. ListRecords ( Resumption)
  6. ListSets ( Resumption)
